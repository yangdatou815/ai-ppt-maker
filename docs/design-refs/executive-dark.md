# Template: executive-dark

> 深色商务风。面向公司年会、产品发布、高管对外汇报这类"灯一关、屏一亮、所有人盯着大屏"的场景。

## 1. 适用场景

- 高管对外宣讲、年度战略发布。
- 大屏 / 投影仪环境（深色更稳，反光弱）。
- 内容偏宏观叙事、品牌感、情绪渲染。

## 2. 调色板

| 角色 | 颜色 | Hex | 备注 |
|---|---|---|---|
| Primary | 深墨蓝 | `#0B1F3A` | 全屏背景 / 大色块 |
| Primary-2 | 午夜黑 | `#050B14` | 章节过渡背景 |
| Accent | 古铜金 | `#C9A24E` | 强调线、关键数字、章节编号 |
| Neutral | 暖米白 | `#F4F1EA` | 正文文字、卡片背景 |
| Mute | 银灰 | `#A3A6AD` | 次要信息、注释 |
| Danger | 砖红 | `#B14B4B` | 仅用于负面数据，慎用 |

> 禁止：纯 `#FFFFFF`（在深色底上太刺眼，用 `#F4F1EA`）、彩虹渐变、霓虹色。

## 3. 字体

- 标题：**思源宋体 CN Heavy**（Source Han Serif CN Heavy）—— 庄重、留得住目光。
- 副标题：思源黑体 CN Medium。
- 正文：思源黑体 CN Regular。
- 等宽 / 数据：JetBrains Mono Regular。

字号：见 README 通用基线。

## 5. 灵感参考

- Apple Keynote 历年发布会黑色封面。
- Bloomberg Terminal 配色（深蓝 + 琥珀）。
- 《经济学人》封面排版的留白与字号对比。

> 仅作视觉灵感，不直接复制任何受版权保护的素材。

## 6. Do / Don't

Do
- 大留白，让金线和大标题成为视觉锚点。
- 关键数字用古铜金 + 等宽字体，强化"权威感"。
- 章节过渡页给到 100% 的视觉冲击（满屏深底 + 巨号编号）。

Don't
- 不要在深底上叠多于 2 种彩色（除主色 / 强调色 / 中性色外）。
- 不要用 PowerPoint 自带的 SmartArt 渐变图标。
- 不要在正文页堆超过 5 条 bullet。
