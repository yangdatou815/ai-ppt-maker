# Template: tech-blue

> 科技蓝风。面向技术发布、SaaS 路演、研发汇报——理性、清晰、有"产品力"质感。

## 1. 适用场景

- 技术架构 / 系统设计汇报。
- SaaS / 开发者工具的产品发布。
- 数据驱动的研发例会、季度 review。

## 2. 调色板

| 角色 | 颜色 | Hex | 备注 |
|---|---|---|---|
| Primary | 科技蓝 | `#1F6FEB` | 标题强调 / 关键控件 |

| Grid | 网格淡蓝 | `#DCE3EE` | 表格 / 图表网格线 |

## 3. 字体

- 标题：**Inter ExtraBold**（英文）/ **思源黑体 CN ExtraBold**（中文）。
- 副标题：Inter SemiBold。
- 正文：Inter Regular / 思源黑体 CN Regular。
- 等宽 / 数据 / 代码：JetBrains Mono Regular（可加亮）。

字号：见 README 通用基线。代码块 / 数据列优先 Mono。

## 6. Do / Don't

Do
- 关键数字 + Mono 字体 + 青绿强调，建立"工程审美"。
- 章节过渡用深空蓝满屏 + 巨号编号，节奏感强。
- 代码块（如有）用 `#0D1117` 底 + 米白 Mono，仿 GitHub 暗色。

Don't
- 不要把科技蓝铺满整页（会变成 PowerPoint 默认蓝感）。
- 不要混入紫色 / 玫红等"AI 风"渐变。
- 不要在数据页用饼图堆叠 —— 优先柱形 / 折线。
