# Template: minimal-light

> 极简白底风。面向投资路演、咨询提案、面向客户的提案文档——干净、可信、阅读友好。

## 1. 适用场景

- 创业 BP / 投资路演 deck。
- 咨询公司 / 设计公司提案。
- 同时需要"上台讲" + "PDF 邮件发出去自己读"的双场景。

## 2. 调色板

| 角色 | 颜色 | Hex | 备注 |
|---|---|---|---|
| Primary | 米白 | `#FAFAF7` | 全屏背景，比纯白柔 |
| Primary-2 | 纯白 | `#FFFFFF` | 卡片 / 表格底 |
| Text | 炭灰 | `#1A1A1A` | 主文字 |
| Text-Mute | 中灰 | `#6B6B6B` | 次要信息 |
| Accent | 朱砂橙 | `#E2532E` | 强调（用量 ≤ 10%） |
| Hairline | 浅灰 | `#E5E2DA` | 1px 分割线、卡片边 |

> 禁止：彩色渐变、超过 1 种强调色。

## 3. 字体

- 标题：**Inter Bold**（英文）/ **思源黑体 CN Bold**（中文）。
- 副标题：Inter Medium / 思源黑体 CN Medium。
- 正文：Inter Regular / 思源黑体 CN Regular。
- 等宽：JetBrains Mono。

字号：见 README 通用基线。

### content-image（上文下图 / 左文右图皆可）

- 图片四周加 1px `#E5E2DA` 发丝线 + 圆角 8px，避免"贴底"。
- caption 居中，中灰 14pt。

### content-table

- 表头：`#1A1A1A` 文字 + 1px 底线。
- 表体：`#FAFAF7` / `#FFFFFF` 交替斑马（极淡）。
- 关键数字列加粗，并允许用朱砂橙强调单元格。

### closing

- 全屏米白底，居中"Thank You"炭灰 64pt Bold + 朱砂橙 ──。
- 下方联系方式中灰 16pt。

## 6. Do / Don't

Do
- 大量留白 + 1px 发丝线建立秩序感。
- 朱砂橙仅在"封面副标短线 / 章节短线 / 关键数字 / 关键 bullet 项目符号"4 处出现。
- 表格用极淡斑马 + 加粗关键列，避免"灰扑扑全表"。

Don't
- 不要叠投影 / 阴影 / 光晕，会破坏极简感。
- 不要用纯黑 `#000000` 文字（太硬，用 `#1A1A1A`）。
- 不要在一页里同时使用图片 + 表格 + 长 bullets，做减法。
